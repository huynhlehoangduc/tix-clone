# TixClone
#### Dự án đươc clone từ trang http://tix.vn/
*Một dự án quản lý rạp chiếu phim*    
Link demo: https://huynhlehoangduc.github.io/tix-clone/  
## Các chức năng:
1. Home page:
- Trang chủ
- Chi tiết phim
- Đặt vé
- Thông tin cá nhân
2. Admin page:
- Quản lý phim - CRUD, Tạo lịch chiếu cho phim
- Quản lý người dùng - CRUD
3. Auth page:
- Login
- Signin
